# Library Inventory

## Submodules
* `JUCE`: Audio and DSP framework (C++). Location: `submodules/juce`. Mapped via `BobQJuceHost`.
* `Ultimate++`: GUI framework (C++). Location: `submodules/ultimatepp`. Mapped via `BobQUltimatePPHost`.

## Go Dependencies
* `gioui.org` (v0.9.0): Primary Go UI rendering backend.
* `github.com/gorilla/websocket` (v1.5.3): Networking.
* `modernc.org/sqlite` (v1.48.1): Database management.

## Qt (Forked)
* `qtbase` components heavily modified and vendored as `BobUI6`/`BobQ`.
